# Caleb Lamoreaux — Digital Portfolio

Personal portfolio site for Caleb Lamoreaux. Built with Next.js 15, shadcn/ui, Tailwind CSS, and Framer Motion.

## Quick start

```bash
pnpm install
pnpm dev
```

Open <http://localhost:3000>.

## Stack

- Next.js 15 (App Router)
- React 19
- shadcn/ui + Radix primitives
- Tailwind CSS 3
- Framer Motion
- TypeScript

## Structure

- `app/` — App Router pages (home, projects, experience, contact)
- `components/ui/` — shadcn/ui primitives
- `components/` — page-level sections and composites
- `lib/` — utilities and project data
- `public/` — static assets and resume PDF
